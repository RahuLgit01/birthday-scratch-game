# Happy Birthday Scratch Game

Wish your friend, cousin, lover or whoever you want a happy birthday like a pro!

## How to customize

Edit the `configuration.js` file to add as many games as you want. You need at least:

- **imagePath** : the picture to scratch
- **backgroundImagePath** : the background behind the message when the image is scratched
- **message** : the message (either it's a win or a loss)

```js
games: [
    {
        imagePath: "images/my-scratch-image.png",
        backgroundImagePath: "images/orange-opacity.png",
        message: "Oh my gosh, Am i right na! Overthinker"
    },
]
```

## How to run

Just open `index.html` in your browser, or host it on GitHub Pages.
